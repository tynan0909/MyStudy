SQL Injection

SQL 인젝션은 애플리케이션이 데이터베이스에 전달하는 SQL 문장에 공격자가 조작된 입력 값을 삽입해, 
원래 의도와 다른 SQL이 실행되도록 만드는 보안 취약점입니다. 
주로 웹 폼, URL 파라미터, 쿠키 등 외부 입력을 적절히 검증하지 않고 SQL에 직접 결합할 때 발생합니다
