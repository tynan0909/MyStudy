13-1 1차원 배열의 이해와 활용

타입이 같은 둘 이상의 데이터를 저장할 수 있는 1차원 구조의 메모리 공간
배열(配列 はいれつ）- 주로 컴퓨터 과학에서 데이터나 요소들을 순서대로 나열하거나 정리하는 것을 의미한다.

int a[ ]; <-- [ ]안에 든건 정수이고, 여러개를 저장한다( int a가 숫자들을 참조하는거임) - 그냥 메모리 많이 잡아먹는다 치면 참조임
int [ ] a = new int[11];
int[ ] a2 = new int[ ] {2,5,4,7}; 는 데이터를 넣는거?
int[ ] a3 = {2,5,4,7}; 도 가능

void a( ) {
	a[10] = 2;
	여기서 10은 데이터가 아니라 인덱스
}
int a2 = a[10];은 a의 10번칸의 값을 a2에 저장하는거	
?.length = 배열의 길이만큼 숫자를 반환해 주는 메소드
