main 메소드는 프로그램의 시작

코딩은 메소드 안에다 해야 함 (메소드와 메소드 사이에 코딩X)
메소드 명을 통해 메소드를 호출할 수 있다

호출하면서 파라미터(매개변수)를 변수이름(int xxx) 에 줄 수 있고
호출받는 메소드 내에서만 사용되며 로컬변수라고 한다

디버깅에서 오류가 쌓이는걸 스택이라 한다?

만약 hiEveryone(12, 12.5)로 되어 있으면
숫자를 두개 줄 수 있다
public static void hiEveryone(int age, double height)의 age, height는 12, 12.5다

return의 의미는 메소드를 호출한 영역으로 값을 반환
public static int adder(int num1, int num2)의 의미는 adder라는 메소드는 num1, num2를 받으면 다른 숫자를 준다는것이다

숫자를 붙이지 않은 return은 메소드의 종료, 탈출을 의미한다
자바프로그램 강제 종료 코드는 System.exit(0)이다 이것은 프로그램을 종료한다(보통 어플만들때 사용한다 함부로 쓰면안댐)

6-3 메소드의 재귀 호출

재귀함수란 정의단계에서 자신을 재참조하는 함수를 뜻한다
ex) 3!(factorial) = 3*2*1 인데 이걸 코드로 만들면
public staticd int factorial(int n) {
	if(n ==1)
		return 1;
	else
		return n * factorial(n - 1);
	}
}

n이 1이 될때까지 계속 호출한다
