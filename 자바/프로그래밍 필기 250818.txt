상수

상수는 모두 대문자로 짓고 단어사이에 언더바 넣음

Literal은 하드코딩

숫자에 아무것도 안붙이면 10진수(왠만하면이걸로씀), 앞에 0붙이면 8진수, 앞에 0x붙이면 16진수(통신할때씀)

역슬래시( ＼ )는 특수한문자로 인식시키는 escape문자임.
ex) ＼t는 tab ＼b는 백스페이스 ＼＼는 백슬래시 ＼r은 캐리지리턴

int는 작은 숫자, long은 큰 숫자임(끝에L붙임)
int num1 = 50;
long num2 = 123456789L;
int num3 = num1 + num2는 불가능!
원래 int와 long 사이에 계산은 안되지만
int num3 = (int) (num1 + num2); 은 뒤를 자르고 강제로 int로 만듬 괄호가 잇는걸 명시적 형변환이라고 함
long num3 = num1 + num2; 으로 하는걸 자동 형변환이라고 함
